# LSR Building - Website Versies

## 📦 Structuur

### `website/` - V1 (Production)
**Status:** ✅ Live op GitHub Pages  
**Branch:** `main`  
**Beschrijving:** Huidige professionele site - focus op kernfunctionaliteit en informatie

**Features:**
- Responsive design
- Contact formulier
- Projectenlijst
- Services overzicht
- SEO basis

---

### `website-v2/` - V2 (Conversion Optimization)
**Status:** 🚀 In Development  
**Branch:** `v2-conversion-optimized`  
**Beschrijving:** Geavanceerde versie met focus op conversie, engagement en enterprise-feel

**Geplande Improvements:**
1. **Conversie Optimalisatie**
   - Sticky CTA widget (floating "Offerte Aanvragen")
   - Exit-intent popup met email capture
   - Multi-step form (reduce abandonment)

2. **Performance & SEO**
   - Image optimization + lazy loading
   - Schema.org markup (FAQSchema, LocalBusiness)
   - Optimized title tags & meta descriptions
   - Internal linking strategy

3. **Trust & Engagement**
   - Case studies met metriek (€ saved, time saved)
   - Team photos + bios
   - Testimonials section
   - Live social proof widgets

4. **Visual Hierarchy & UX**
   - Prioritized services ranking (top 3 featured)
   - Before/after project sliders
   - Animated hero section
   - Micro-interactions (hover effects, scroll animations)

5. **Backend Enhancement**
   - Lead scoring system
   - Email automation sequences
   - CRM integration ready (HubSpot/Pipedrive)

## 🚀 Hoe te Gebruiken

### V1 Starten (Production)
```bash
cd website
npm install
npm run dev
# http://localhost:3000
```

### V2 Starten (Development)
```bash
cd website-v2
npm install
npm run dev
# http://localhost:3001 (of ander poort)
```
